## 创建项目

### 下载必要的依赖



```
npm install less

npm install vue-router
npm install element-plus
npm install @element-plus/icons-vue
```



### 配置@别名

引入一个其它，@符号定位到src目录

在vite.config.js下

```
export default defineConfig({
  plugins: [vue()],
  // 添加的别名
  resolve: {
    alias:[
      {
        find:"@",
        replacement:"/src",
      },
    ]
  }
})
```
